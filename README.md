# P1-ai-resume-analyzer

🚀 AI Resume Analyzer & Job Recommender

An intelligent web application that analyzes resumes and suggests suitable job roles using basic Natural Language Processing (NLP) and data matching techniques.

📌 Overview

This project allows users to upload their resume in PDF format and get instant job recommendations based on the skills detected in the resume.

It extracts text from the resume, identifies relevant skills, and compares them with job descriptions to suggest the best matching roles.
